My Android projects

Calculator v2

Calculator v2 is an advanced calculator with functions such as trigonometry, logs, factorials, and more while still maintaining a simple and easy to use interface.

Card Cliques

Card Cliques is a fun and challenging card game in which players find three cards where each trait (color, shape, fill, or number) is all different or all the same.

Quackcryption

Quackcryption uses 128 bit AES encryption to turn standard text into an encrypted string. It won the prize for Best Developer Tool at SpartaHack 2016. It's "unquackable"!

JavaScript things

Nix.js

Nix.js is an OS for your browser, similar to UNIX and written entirely in JavaScript. It has a virtual file system, a package manager, and a CLI.

Truth Table Generator

Truth Table Generator is a web app to help users make truth tables quickly and easily. It is intended to help people building circuits with logic gates.

My TI 84 stuff

TI Games

Wouldn't it be great if you could play your favorite smartphone games on your graphing calculator? Now you can, and math class will never be boring again.

AlphaCS

AlphaCS edits the Virtual Allocation Table to unlock hidden features such as hiding and locking programs. It uses zStart's API to serve as a fully functional launcher.

TI Terminal

This app uses inline assembly to fetch data that would otherwise be unavailable, such as battery life, hidden files, free space, and a full set of program management features.
