My Android projects

Card Cliques

Card Cliques is a fun and challenging card game in which players find three cards where each trait (color, shape, fill, or number) is all different or all the same.

Quackcryption

Quackcryption uses AES encryption to turn standard text into an encrypted string. It won the prize for Best Developer Tool at SpartaHack 2016. It's "unquackable"!

Calculator v2

Calculator v2 is an advanced calculator with functions such as trigonometry, logs, factorials, and more while still maintaining a simple and easy to use interface.

JavaScript things

Truth Table Generator

This web app helps users make truth tables quickly and easily. It's useful when building circuits from logic gates.

Nix.js

Nix.js is a JS-based, UNIX-like OS that lives in your browser. It has a CLI, a virtual file system, and a package manager.

My TI 84 stuff

TI Games

Wouldn't it be great if you could play your favorite smartphone games on your graphing calculator? Now you can, and math class will never be boring again.

AlphaCS

AlphaCS edits the Virtual Allocation Table to unlock hidden features such as hiding and locking programs. It integrates with zStart to serve as a program launcher.

TI Terminal

This app uses inline assembly to fetch data that would otherwise be unavailable, such as battery life, hidden files, RAM usage, and a full set of program management features.
