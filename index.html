A little bit about me

I'm Scott Mangiapane, a software engineer in Chicago, IL. This website is primarily a place for me to showcase my personal projects, so it leans heavily on the software-engineering side. But I have other hobbies as well... Swimming, tennis, piano, chess, and leathercraft to name a few. Most recently I've been learning the bass guitar.

My engineering journey

I started programming on my TI-84 graphing calculator back in high school. I didn't have a smartphone at the time, so I made my own versions of popular phone games on the calculator and distributed them to my classmates (to the dismay of my math teachers). I've come a long way since then, but have maintained that same enthusiasm. I've spent the last 5 years as a full-stack engineer at GoDaddy building the API and dashboard for their VPS (virtual private server) product. Most recently I've been improving my Go and AWS skills.

Web apps

Podfish

A free, minimal, and self-hostable podcast client. Features a Go backend and React SPA frontend. Leverages Docker. Currently under active development.

CalcNotes

calcnotes.app has an interface made for desktop computing while also providing advanced features like variables and custom functions.

DB Manager

Database Manager is a cross-platform desktop app that lets users browse the contents of PostgreSQL databases and run custom queries.

Android stuff

Card Clicks

Card Clicks is a fun and challenging card game in which players find three cards where each trait (color, shape, fill, or number) is all different or all the same.

Quackcryption

Quackcryption uses AES encryption to turn standard text into an encrypted string. It won the prize for Best Developer Tool at SpartaHack 2016. It's "unquackable"!

Calculator v2

Calculator v2 is an advanced calculator with functions such as trigonometry, logs, factorials, and more while still maintaining a simple and easy to use interface.

TI 84 programs

TI Games

Wouldn't it be great if you could play your favorite smartphone games on your graphing calculator? Now you can, and math class will never be boring again.

AlphaCS

AlphaCS integrates with zStart to serve as a program launcher. It edits the Virtual Allocation Table to unlock hidden features such as hiding and locking programs.

TI Terminal

A full program manager with a command prompt user interface. It includes many useful commands like checking the battery status and toggling TI-OS lowercase.
